# Demo Agent Scaffold

This module includes `Agents/DemoAgent` as a **template only**.

Do not treat it as a required booking/production agent. For a real module, copy the pattern and create a module-specific agent such as:

- `booking.agent`
- `estimating.agent`
- `contracts.agent`
- `compliance.agent`

The user-facing AI name should remain **Zero** everywhere. Specialist agents stay internal.
