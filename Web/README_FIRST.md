# QRPay Web Broader Extraction v2

This pack broadens the prior extraction beyond payment-link and QR basics.

- App core and config/provider surfaces
- Merchant advanced flows
- Gateway checkout flows

- Developer API surfaces

- Routes, config, and migrations
- Shared views and assets

Use this as a broader donor-code pack for ZeroPay and adjacent products.
