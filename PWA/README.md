# ZeroPay PWA

Progressive Web App shell for the ZeroPay payment platform.

## Tech Stack

- React 18 + Vite + TypeScript
- `vite-plugin-pwa` (Workbox service worker)
- `react-router-dom` v6
- `idb` (IndexedDB)
- `axios` (API)

## Setup

```bash
npm install
cp .env.example .env
# Edit .env with your API base URL and VAPID public key
npm run dev
```

## Build

```bash
npm run build
# Output: dist/
```

## PWA Features

| Feature | Strategy |
|---|---|
| Shell (HTML/CSS/JS) | Cache-first (pre-cached by Workbox) |
| API calls (`/api/*`) | Network-first, 10 s timeout, falls back to cache |
| Google Fonts | Cache-first, 1 year TTL |
| Offline fallback | `public/offline.html` served when navigation fails |
| Offline payments | Background Sync — queued in IndexedDB, replayed on reconnect |
| Push notifications | Web Push — VAPID, deep-link on click |

## Service Worker

The service worker is generated by VitePWA / Workbox from `vite.config.ts`.
Custom behaviour (background sync, push handling) lives in `src/sw.ts`.
